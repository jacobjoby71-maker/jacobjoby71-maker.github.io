All four channels feed into a single Google Sheet via Apps Script · Trustee Console reads & writes to the same sheet

All three public channels (website, Facebook, WhatsApp) share a single form URL with a ?source= parameter that tags each submission. The trustee kiosk submits directly. A Google Apps Script web app acts as the API bridge — receiving POST requests from all channels and writing rows to one shared Google Sheet. The Trustee Console reads from that same sheet, providing a unified real-time view.

RECOMMENDED BACKEND: GOOGLE SHEETS

✝ OUR RECOMMENDATION — GOOGLE SHEETS + APPS SCRIPT

💰

Completely free. Google Sheets and Apps Script have no cost for a church's usage volume. No monthly fees, no server costs.

📊

Familiar to all trustees. Data lives in a spreadsheet that any trustee can open on any device. No special software, no training needed.

🔒

Secure & access-controlled. Share the Google Sheet only with the finance committee. Google handles authentication and encryption.

🔄

Always in sync. Every submission from any channel appears in the sheet instantly. Year-end totals, receipts, and reports are always current.

📤

Easy to export. Download as CSV or Excel for the church accountant. Works with all financial software used by Canadian churches.

🆚
vs. Airtable
Airtable has a better UI but costs $20/month for more than 1,000 records. Not worth the expense for a church. Google Sheets is unlimited and free.
🆚
vs. Supabase / Firebase
These require a developer to set up and maintain a database. Google Sheets requires zero infrastructure — any trustee can manage it.
🆚
vs. Stripe / PayPal only
Payment platforms only capture online payments. Google Sheets captures all channels including cash, cheque, and kiosk — essential for a complete picture.
⬆
Future upgrade path
If the church grows, data can be migrated from Google Sheets to a proper database (Supabase, Postgres) with no changes to the forms.

ONE-TIME SETUP — STEP BY STEP
1

Create the Google Sheet

Go to sheets.google.com · Create a new spreadsheet · Name it "TMC Vancouver Contributions" · Add a sheet tab called Contributions. Note the Sheet ID from the URL (the long string between /d/ and /edit).

2

Open Apps Script

In your Google Sheet, go to Extensions → Apps Script. Delete any existing code. Open the Trustee Console → Settings tab and copy the provided Apps Script code. Paste it into the editor.

3

Enter your Sheet ID

Replace YOUR_GOOGLE_SHEET_ID_HERE in line 1 of the script with your actual Sheet ID from Step 1. Click Save (Ctrl+S).

4

Deploy as a Web App

Click Deploy → New Deployment. Set type to Web App. Set "Execute as" to Me. Set "Who has access" to Anyone. Click Deploy. Copy the web app URL.

5

Paste URL into Trustee Console

Open the Trustee Console → Settings → paste the Apps Script URL into the Google Sheets Endpoint field. Click Save. All future entries will sync automatically.

6

Update all three public forms

Open each of the three public HTML files (contribution.html, contribution-facebook.html, contribution-whatsapp.html). Add one line near the top: const ENDPOINT = 'YOUR_APPS_SCRIPT_URL'; and update each form's submit handler to POST to that endpoint with a channel field.

7

Share the sheet with trustees

In Google Sheets, click Share and add the church treasurer's and finance committee's emails with Viewer or Editor access. They can view contributions at any time without the kiosk app.

STANDARD OPERATING PROCEDURES

SUNDAY COLLECTION DRIVE (KIOSK)

Weekly
  • Before service: Open Trustee Console on tablet, ensure Wi-Fi is connected (green sync dot). Verify PIN works.
  • During collection: Hand tablet to each member. They enter their name; trustee enters amounts as told. Tap "Save Entry" after each member.
  • Cash/cheque: Record in Trustee Notes field (e.g. "Cash $200 — Ref: Envelope #14"). Keep physical envelopes numbered.
  • After service: Tap "📥 Export CSV" and email to treasurer. Verify sync dot is green (all entries uploaded to Sheets).
  • Lock the console before handing the tablet back to be stored.

MONTHLY DIGITAL CAMPAIGN (WEB / FB / WA)

Monthly
  • First Sunday of month: Open the Facebook page link → copy the post text → paste into the church Facebook group. Replace [PASTE YOUR LINK HERE] with the hosted contribution URL.
  • Same day: Open the WhatsApp page → tap "Open in WhatsApp" → forward to all church WhatsApp groups and the community group.
  • Pin the post in the Facebook group and WhatsApp community so it stays visible.
  • Follow up mid-month: Open Trustee Console → All Entries → filter by Web/Facebook/WhatsApp to see who has submitted online this month.
  • Reminder message: Send a gentle reminder via WhatsApp to groups 2 weeks in, noting the online contribution link.

MONTHLY RECONCILIATION (TREASURER)

End of Month
  • Open Google Sheets → Contributions tab · Filter by the current month's date column.
  • Cross-reference PayPal transactions (log into PayPal, download CSV) against entries marked "PayPal" in Notes column.
  • Cross-reference Interac e-Transfer bank notifications against entries with "e-transfer" in Notes.
  • Cross-reference cash/cheques deposited against kiosk entries with Cash/Cheque in Notes.
  • Any discrepancy → check the member's Ref No. and contact them directly via the phone/email in the sheet row.
  • Create a monthly summary: File → Download → .xlsx for the church accounting records.

YEAR-END TAX RECEIPTS (FINANCE COMMITTEE)

Annual
  • In Google Sheets: use SUMIF formula to total contributions per member (group by Name + Ref No).
  • Export master list as CSV → import into your receipt-generating software (or use a mail-merge template in Word).
  • CRA requires receipts for all contributions over $10. Receipts must show: church name, charity registration number, donation amount, and date.
  • Email receipts to members whose email is captured in the sheet. Mail paper receipts to those without email.
  • Archive: Download the full year's Google Sheet as Excel and store on church OneDrive/Google Drive for 7 years (CRA requirement).

SECURITY & ACCESS CONTROL
🔐
Change the Default PIN
The default PIN is 1234. Change it immediately in Trustee Console → Settings → Change PIN. Use a 4-digit code known only to trustees. Change it if any trustee leaves the role.
📋
Google Sheet Access
Share the Google Sheet only with the treasurer and finance committee. Use "Viewer" access for committee members and "Editor" only for the treasurer.
📱
Tablet / Device Security
Enable device PIN/passcode on the tablet. Use "Guided Access" (iOS) or "App Pinning" (Android) to lock the browser so members can't navigate away from the kiosk.
⚠
No Sensitive Data
The system does not store credit card numbers. PayPal handles all card processing. For e-transfers, only the amount and name are recorded — no banking details.
🍁
PIPEDA Compliance
Under Canadian privacy law, inform members their contact info is stored for contribution tracking and tax receipts only. Add a brief notice at the top of the contribution form.
